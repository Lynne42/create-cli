Initialize typescript project based on umi-react-antd-pro, including basic code style check eslint, prettier, etc.

# 依赖
npm 9.2.0
umi 4.0.40
react 18.0.0
typescript 4.1.2
tailwindcss 3.2.4
antd 5.0.0

# 使用
```
npx @qln/umi-ts-pro  project-demo
```
选择使用antd pro模板

# 开发
```
npm run dev
```

# build
```
npm run build
```
