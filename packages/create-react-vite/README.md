Initialize typescript project based on vite-react-ts, including basic code style check eslint, prettier, etc.

# 依赖
yarn 3.3.1
vite 4.0.0
react 18.2.0
typescript 4.9.3
tailwindcss 3.2.4
antd 5.1.0

# 使用
```
npx @qln/vite-react-ts project-demo
```

# 开发
```
yarn dev
```

# build
```
yarn build
```
